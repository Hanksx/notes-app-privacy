Política de Privacidade - Notes App

Última atualização: 02 de dezembro de 2025

1. Introdução

O Notes App ("nós", "nosso" ou "app") respeita sua privacidade e está comprometido em proteger seus dados pessoais. Esta política de privacidade explica como coletamos, usamos e protegemos suas informações.

2. Informações que Coletamos

2.1 Dados Armazenados Localmente

  • Notas: Todo o conteúdo das suas notas é armazenado localmente no seu dispositivo
  • Configurações: Preferências do app são salvas localmente
  • Chat P2P: Mensagens de chat são transmitidas diretamente entre dispositivos sem armazenamento em servidores

2.2 Dados Coletados por Terceiros

  • Google AdMob (versão gratuita): Pode coletar identificadores de dispositivo e dados de uso para exibir anúncios personalizados
  • Firebase (opcional): Se você usar recursos de sincronização, dados podem ser armazenados no Firebase
  • Google Play Billing: Informações de compra são processadas pelo Google Play

3. Como Usamos Suas Informações

  • Fornecer funcionalidades do app (notas, chat P2P)
  • Processar compras in-app (Premium/Pro)
  • Exibir anúncios (apenas versão gratuita)
  • Melhorar a experiência do usuário

4. Compartilhamento de Dados

Não compartilhamos, vendemos ou alugamos seus dados pessoais.

Dados podem ser compartilhados apenas com:

  • Google AdMob (para exibição de anúncios)
  • Google Play (para processar compras)
  • Firebase (se você optar por usar sincronização)

5. Chat P2P (Peer-to-Peer)

  • Mensagens são transmitidas diretamente entre dispositivos
  • Não armazenamos mensagens em nossos servidores
  • Mídia (fotos, vídeos, áudios) é enviada via Firebase apenas para usuários Premium/Pro
  • Você é responsável pelo conteúdo compartilhado no chat

6. Segurança

Implementamos medidas de segurança para proteger seus dados:

  • Armazenamento local criptografado
  • Conexões seguras (HTTPS/TLS)
  • Sem armazenamento de senhas (autenticação via Google)

7. Seus Direitos

Você tem direito a:

  • Acessar seus dados
  • Excluir seus dados (desinstalar o app)
  • Desativar anúncios personalizados (configurações do dispositivo)
  • Cancelar assinatura Premium a qualquer momento

8. Crianças

Nosso app não é direcionado a menores de 13 anos. Não coletamos intencionalmente dados de crianças.

9. Alterações nesta Política

Podemos atualizar esta política periodicamente. Notificaremos sobre mudanças significativas através do app.
